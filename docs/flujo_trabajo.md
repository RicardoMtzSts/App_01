# Flujo de Trabajo de GraphOS

## 1. Creación de Proyecto
- El usuario crea una carpeta de proyecto desde la app.
- Puede anidar carpetas y documentos ilimitadamente.

## 2. Selección de Organizador Gráfico
- Elige entre los tipos soportados (mapa mental, diagrama de flujo, cuadro comparativo, etc.).
- Se carga la plantilla correspondiente desde /src/organizers.

## 3. Edición y Personalización
- Añade figuras, conectores, texto, imágenes, tablas, gráficos, etc.
- Personaliza colores, fondos, tamaños y estilos.
- Usa herramientas de zoom, reglas, guías y navegación fluida.

## 4. Gestión de Notas
- Crea notas enriquecidas vinculadas a documentos o globales.
- Usa tags, bloques de código, enlaces y búsqueda avanzada.

## 5. Importación y Exportación
- Importa recursos (imágenes, PDF, CSV, JSON, XML, texto).
- Exporta a PDF, SVG, PNG, JPG, PPTX, JSON/XML.
- Opciones de calidad, selección de área, impresión directa.

## 6. Guardado y Autoguardado
- Guarda automáticamente cada 30 segundos (configurable).
- Historial de versiones y recuperación de cambios.

## 7. Accesibilidad y Personalización
- Modo daltonismo, atajos de teclado, temas visuales.
- Preferencias de usuario y autocorrección ortográfica.

## 8. Sincronización y Multi-ventana
- Soporte para múltiples ventanas y documentos abiertos.
- Sincronización futura con nube (Google Drive, OneDrive, Dropbox).

---

Este flujo asegura una experiencia de usuario flexible, potente y adaptable a distintos tipos de organizadores gráficos y necesidades educativas/profesionales.
